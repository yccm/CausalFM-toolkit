Installation
============

Requirements
------------

* Python 3.8+
* PyTorch 2.0+
* CUDA support (optional, for GPU acceleration)

Install from Source
-------------------

Clone the repository and install dependencies:

.. code-block:: bash

   git clone https://github.com/yourusername/CausalFM-toolkit.git
   cd CausalFM-toolkit
   pip install -r requirements.txt

Verify Installation
-------------------

.. code-block:: python

   import causalfm
   print("CausalFM Toolkit installed successfully!")
