CausalFM Toolkit Documentation
===============================

Welcome to CausalFM Toolkit's documentation!

CausalFM Toolkit is a comprehensive framework for causal inference using foundation models.

.. toctree::
   :maxdepth: 2
   :caption: Contents:

   installation
   quickstart
   api/index

Features
--------

* Standard causal effect estimation
* Instrumental Variable (IV) methods
* Front-door adjustment
* Integration with TabPFN models

Installation
------------

.. code-block:: bash

   pip install -r requirements.txt
